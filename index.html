Portfolio Project

My first project in this repository!

I started this project immediately after completing Shay Howe's "Learn to Code HTML & CSS" guide, and I tried to incorporate some of the things that I learned in that tutorial. A small sample of new-to-me items:

  • Using labels in form elements
  • Using HTML comments to display "inline-block" elements on the same line
  • Using HTML5 tags like "section" and "header"
  • Using linear-gradient to style a button

I'm not entirely happy with this page. Obviously the content isn't serious, but I think that there is also too much white (the color), and the site doesn't work real well on small screens. That said, I'm going to go ahead and call this one finished for now - I can always go back and modify it later.

My favorite part? Probably the styling of the "Firstname Lastname" in the header. I used two shadows, and I think it turned out looking pretty neat.
